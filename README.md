### Quick fixes / Must knows

| Action           | Keys          | Context       | 
| ---------------- | ------------- |:-------------:|
| Quick fix        | ⌥ ENTER       |               | 
| Organize import  | ⌃ ⌥ O         |               |  
| Indent lines     | ⌃ ⌥ I         |               |
| Reformat code    | ⌥ ⌘ L         |               | 
| New Class        | ⌘ N           | Project  tab  | 
| Generate code    | ⌘ N           | Java File     | 
| Show docs        | ⌃ Q           |               | 
| Action look-up   | ⇧ ⌘ A         |               |
| Run              | ⌃ R           |               |

| Go to type declaration  | ⌃ ⇧ B         |               |

| Go to test              | ⇧ ⌘ T         |               |

| Go forward              | ⌥ ⌘ →         |               |
